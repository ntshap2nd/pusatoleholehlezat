## üîß Yang Perlu Dilakukan:

### 1. **Setup Google Analytics 4**

**Step 1:** Buat GA4 Property
1. Buka https://analytics.google.com
2. Create Account ‚Üí Create Property
3. Property name: "Pusat Oleh-Oleh Lezat"
4. Pilih Indonesia timezone
5. Copy Measurement ID (format: G-XXXXXXXXXX)

**Step 2:** Replace Placeholder ID
Edit `index.html` line ~189:
```html
<!-- BEFORE -->
gtag('config', 'G-XXXXXXXXXX', {

<!-- AFTER -->
gtag('config', 'G-YOUR_REAL_ID', {
```

Edit `src/lib/analytics.ts` line ~34:
```typescript
// BEFORE
window.gtag!('config', 'G-XXXXXXXXXX', {

// AFTER
window.gtag!('config', 'G-YOUR_REAL_ID', {
```

### 2. **Setup Facebook Pixel (Optional)**

**Step 1:** Create Meta Pixel
1. Buka https://business.facebook.com
2. Events Manager ‚Üí Create Pixel
3. Copy Pixel ID

**Step 2:** Uncomment & Update
Edit `index.html` line ~206-207:
```html
<!-- BEFORE (commented out) -->
// fbq('init', 'YOUR_PIXEL_ID');
// fbq('track', 'PageView');

<!-- AFTER -->
fbq('init', '1234567890123456');  // Your actual pixel ID
fbq('track', 'PageView');
```

Uncomment line ~209:
```html
<!-- BEFORE -->
<!-- <noscript><img ... /></noscript> -->

<!-- AFTER -->
<noscript><img height="1" width="1" style="display:none" 
  src="https://www.facebook.com/tr?id=1234567890123456&ev=PageView&noscript=1" />
</noscript>
```

Edit `public/sitemap.xml`:


## üìä How to Use Analytics

### Track Events in Components:

```typescript
import { trackAddToCart, trackWhatsAppClick, trackSearch } from '@/lib/analytics';

// When adding product to cart
trackAddToCart({
  id: product.id,
  name: product.name,
  category: product.category,
  price: product.price,
  quantity: 1
});

// When clicking WhatsApp button
trackWhatsAppClick('checkout');

// When searching
trackSearch(searchQuery, resultsCount);
```
